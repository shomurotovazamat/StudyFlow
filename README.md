# StudyFlow — Умный помощник студента

**Проект:** StudyFlow | **Дисциплина:** Управление инновационным проектом  
**Команда:** Шомуротов Азамат · Хтет Аркар Кяв

---

## Описание

StudyFlow — веб-приложение для студентов, которое объединяет расписание занятий, трекер задач и систему умных
напоминаний в одном месте. Помогает не пропускать дедлайны и эффективно планировать учебную нагрузку.

---

## Структура проекта

```
studyflow/
├── src/
│   ├── main.jsx                   ← Точка входа
│   ├── App.jsx                    ← Главный компонент (состояние, сайдбар, модалы)
│   ├── index.css                  ← Глобальные стили
│   ├── constants.js               ← Константы, начальные данные, вспомогательные функции
│   ├── hooks/
│   │   └── useLocalStorage.js    ← Хук для сохранения данных в браузере
│   ├── components/
│   │   └── UI.jsx                ← Общие компоненты (Tag, Btn, Modal, Field...)
│   └── pages/
│       ├── TodayView.jsx         ← Страница «Сегодня»
│       ├── ScheduleView.jsx      ← Недельное расписание
│       ├── TasksView.jsx         ← Трекер задач (Kanban)
│       └── NotifView.jsx         ← Уведомления
├── index.html
├── package.json
└── vite.config.js
```

## Установка и запуск

```bash
# 1. Установить зависимости
npm install

# 2. Запустить в режиме разработки
npm run dev

# 3. Собрать для продакшена
npm run build
```

---

## Функциональность

| Страница        | Описание                                                 |
|-----------------|----------------------------------------------------------|
| **Сегодня**     | Актуальные занятия и ближайшие дедлайны, статистика      |
| **Расписание**  | Недельное расписание, добавление и удаление занятий      |
| **Задачи**      | Kanban-доска: Не начата → В процессе → Готово            |
| **Уведомления** | Браузерные push-уведомления за 24 ч и за 1 ч до дедлайна |

---

## Особенности

- **localStorage** — данные сохраняются в браузере, не теряются при обновлении страницы
- **Без авторизации** — приложение работает сразу, без регистрации
- **Бесплатные технологии** — React, Vite, Vercel, GitHub

## Деплой на Vercel (бесплатный хостинг)

**Вариант 1 — через GitHub (рекомендуется):**

1. Зайдите на [vercel.com](https://vercel.com) → **Add New Project**
2. Выберите репозиторий GitHub → **Import**
3. Framework: **Vite** (определяется автоматически)
4. Нажмите **Deploy** — готово за 1 минуту

**Вариант 2 — через терминал:**

```bash
npm install -g vercel
vercel
```
